# What is AI?

কৃত্রিম বুদ্ধিমত্তা (AI) is software that learns patterns from data instead of
following rules a human wrote by hand.

## The short version

A traditional program is a recipe: do this, then that. An AI model is closer to
a student — you show it thousands of examples, and it figures out the rule
itself.

- **Data** — the examples it learns from.
- **Model** — the thing that holds what it learned.
- **Inference** — using the trained model to answer something new.

## Why now?

Three things lined up: cheap compute, huge datasets, and better training
methods. None of these is new on its own; together they crossed a threshold.
